Explain why the alt attribute of the img tag is important
 the alt attribute allows for an alternative description of the image in that particular element. It is useful for browsers that do not support images and for those who are blind and are using a screen reader to navigate the page.
What is the purpose of name mangling?
 Name mangling is used to solve the problems involved with requiring that every element have a unique ID on a page. It guarantees that every element has a unique id
Explain the difference between redirects that use a 301 HTTP status code and ones that use a 302 HTTP status code
 A 301 status code indicates that the website has moved permanently and that the link used to access that website should no longer be used. A 302 status code indicates that the website has only moved temporarily and that the link should be used later to access the website.
